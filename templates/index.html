◆ Imagens de produtos

Busque imagens de produtos pelo código de barras

A aplicação inicia offline. Escolha a fonte: o ERP WinThor (Oracle) ou um arquivo Excel com os códigos a pesquisar.

Informe a conexão com o banco Oracle do WinThor. A conexão só é feita quando você testar ou buscar — e a senha não é salva em disco.

Envie um arquivo .xls ou .xlsx contendo a coluna CODBARRA com os códigos de barras (EAN/GTIN) a pesquisar. As imagens são salvas com o próprio código de barras como nome.

Registro das operações desta sessão (também gravado em load_images.log).
